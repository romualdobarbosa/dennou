## O que faz
Exibe todas as notas do vault como nós e as conexões entre elas via `[[wiki-links]]`.

## Uso
```
Ctrl+G
```
Ou pelo menu: **View → Open graph view**

## Flags úteis
| Recurso | Como usar |
|---|---|
| Filtrar por tag/pasta/texto | Painel lateral do grafo |
| Navegar para uma nota | Clicar no nó |

## Exemplo
Notas com `## Links` preenchido aparecem conectadas no grafo. Notas sem nenhum `[[link]]` aparecem isoladas.
