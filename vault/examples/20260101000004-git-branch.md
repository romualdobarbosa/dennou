## O que faz
Cria, lista e deleta branches no repositório.

## Uso
```bash
git branch <nome>          # criar
git branch                 # listar locais
git branch -a              # listar locais e remotos
git branch -d <nome>       # deletar (seguro)
git branch -D <nome>       # deletar forçado
```

## Flags úteis
- `-d` — deleta só se o branch já foi mergeado
- `-D` — deleta independente do estado
- `-m <novo>` — renomear o branch atual

## Exemplo
```bash
git branch feature/login
git checkout feature/login
# ou em um comando só:
git checkout -b feature/login
```
