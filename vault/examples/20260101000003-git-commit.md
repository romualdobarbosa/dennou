## O que faz
Salva um snapshot das mudanças staged no histórico do repositório.

## Uso
```bash
git commit -m "mensagem"
```

## Flags úteis
- `-m` — mensagem inline
- `-a` — staged + unstaged de arquivos já rastreados (pula o `git add`)
- `--amend` — reescreve o último commit (não usar se já foi publicado)

## Exemplo
```bash
git add src/main.py
git commit -m "feat: adicionar endpoint de login"
```
