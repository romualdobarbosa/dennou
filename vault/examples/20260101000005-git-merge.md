## O que faz
Integra o histórico de um branch em outro.

## Uso
```bash
git merge <branch>
```

## Flags úteis
- `--no-ff` — força um merge commit mesmo quando fast-forward é possível (preserva histórico)
- `--squash` — comprime todos os commits do branch em um único antes de mergear
- `--abort` — cancela um merge em conflito

## Exemplo
```bash
git checkout main
git merge feature/login --no-ff -m "merge: integrar feature de login"
```

## Gotchas
- Conflitos precisam ser resolvidos manualmente — edite os arquivos marcados, faça `git add` e depois `git commit`
- `--squash` não cria o commit automaticamente, só staged as mudanças
