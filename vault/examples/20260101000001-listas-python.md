## O que é
Estrutura de dados ordenada e mutável. Aceita elementos de tipos diferentes.

## Sintaxe
```python
lista = [1, "dois", 3.0]
lista.append(4)
lista.remove("dois")
lista[0]       # acesso por índice
lista[-1]      # último elemento
lista[1:3]     # slice
```

## Exemplo
```python
frutas = ["maçã", "banana", "uva"]
frutas.append("laranja")
print(frutas[0])   # maçã
print(len(frutas)) # 4
```

## Gotchas
- `lista2 = lista` não copia — ambas apontam pro mesmo objeto. Use `lista.copy()` ou `lista[:]`
- `remove()` apaga apenas a primeira ocorrência
